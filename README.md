# Mode Prompts Analysis

This repository contains the analysis of three AI agent mode prompts: **Architect**, **Debug**, and **Ask**.

## Contents

- [mode_prompts_analysis_en.md](mode_prompts_analysis_en.md): Analysis report in English.
- [mode_prompts_analysis_ko.md](mode_prompts_analysis_ko.md): Analysis report in Korean.

## Overview
The analysis compares the roles, permissions, mandatory tools, and constraints of each mode to understand how they shape the agent's behavior.
